To run this project locally, you don't need any complex build tools. It is a static website built with HTML, CSS, and vanilla JavaScript.

### Prerequisites

*   A modern web browser.
*   Python 3 (optional, for running a local server).

### Running Locally

1.  **Clone the repository:**
    ```bash
    git clone https://github.com/manojnagarajupalle/GTA_Website_v2.git
    cd GTA_Website_v2
    ```

2.  **Start a local server:**
    If you have Python installed, you can simply run:
    ```bash
    python3 -m http.server 8000
    ```

3.  **View the site:**
    Open your browser and navigate to `http://localhost:8000`.
